Import WooCommerce products from any API source, including JSON, XML, or CSV. Field mapping, profile saving, and branded UI included.

== Description ==

This plugin allows you to import WooCommerce products from:
- JSON APIs
- XML endpoints
- CSV files or URLs

Includes:
- Field Mapping UI (like WP All Import)
- Import Profiles
- API Credential Input
- Scheduled Import (WP-Cron)
- GitHub auto-updates
- Custom branding with green/dark green styling

== Installation ==

1. Upload the plugin via WordPress Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Go to the “Piwah API WooCommerce Importer” in the admin sidebar.
4. Enter your API settings, map fields, and click Import.
